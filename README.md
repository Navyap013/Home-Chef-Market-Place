# Home Chef Market Place

A full-stack web application that connects home chefs with local customers looking for fresh, homemade meals.

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript, React.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose)
- **Version Control**: Git, GitHub

##  Features
- Chef Registration & Login
- Menu Listing & Order Placement
- Location-Based Search for Meals
- Order Cart & Checkout Simulation
- Admin/Chef Dashboard
